The Basics of Javascript


How does JavaScript compare to HTML and CSS?

While HTML and CSS are static languages, JavaScript is dynamic. It's like a magic wand that allows the person to interact with, and change the page or app.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

A loop is fairly self-explanitory, a set of instructions that repeat over and over again as long as a set of conditions are met. Once those conditions are no longer met the loop stops. Say you want a glass of milk. It involves several steps in a row, plus you will only do it if you're thirsty and will stop once you're full. If a loop is a repetitive set of actions, then control flow is the order in which those actions happen. Back to pouring your milk. If you pour the milk without unscrewing the lid first then nothing will happen, but if you do them in the right order everything will work fine.

Explain the difference between accessing data from arrays and object literals.

Arrays and Objects are both similar, but have important differences. Arrays are very good when order is needed, every property is kept and accessed from it's exact place in the array, which makes it great for lists but hard to manipulate. Objects on the other hand aren't built for order and use "keys" to access their assigned property.

Explain what functions are and why they are useful.

A function is able take a huge set of rules often many, many lines long and make them equal a single variable. That way, instead of typing out huge amounts of code everytime (taking up precious time and computer memory) the developer can use a couple of words instead. They often take inputs from the user and can have different results depending on what the user writes.
