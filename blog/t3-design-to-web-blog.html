Responsiveness


What is a responsive site, and why is responsiveness important?

A responsive website is one that can adapt it's layout and change it's elements to suit the screen it is being viewed on.

What is mobile first design, and why is it important?

Because the world is becoming more connected and increasingly mobile, phones and tablets are quickly becoming the main devices for browsing the internet. This means it's beneficial to design a responsive website for x-small (or mobile) screens first, and adapt them for small, medium, large and x-large screens after. Mobile first design is taking this approach when going through the design process.

What are frameworks, and what are their pros and cons?

A framework is a pre-written set of css code. It is an easy way to make a website look and feel nice without having to write all the code yourself. There are several frameworks that others have already written so there is plenty of choice with layouts and design. Some offer a full range of pre-coded options eg: buttons, tables, modals, links, navbars, image carousels etc, while others only offer a few. While it may seem better to use a framework that has all the options, there are also benefits in using a more specific, streamlined framework; such as faster loading times and more intimate knowledge of the code and what it's capable of. While frameworks are a nice, easy, lazy way to code they aren't perfect, and often still need tweaking to look and behave exactly how you want.

What is a wireframe and why do we use it?

Imaginge looking at this website (or any website) and trying to recreate it using only chicken wire. You could outline everything and show where stuff goes and how it relates to the rest of the page, but not much more. Wireframing is drawing (hand or using an app) your website as if it's made of chicken wire, basic and without any extra design distractions. It's a great step in the design process and allows you as the designer to get a good feel for how the site will flow and feel to the user.
